NEPAL MILK DAIRY - PROJECT INFORMATION & ACCESS GUIDE

1. PROJECT OVERVIEW
-------------------
Nepal Milk Dairy is a dynamic e-commerce web application for selling organic dairy 
products. The system features separate interfaces for the public, customers, and 
vendors/administrators.

2. FILE STRUCTURE
-----------------
The project is organized into modular directories:

- root/
  - index.php          : Main Landing Page (Contains auto-setup logic for first run)
  - database.sql       : Database structure (tables: users, products, orders, etc.)

- public/              : Public-facing pages
  - login.php          : Customer Login
  - register.php       : Customer Registration
  - products.php       : Product Catalog
  - cart.php           : Shopping Cart
  - checkout.php       : Checkout Process

- admin/               : Vendor/Admin Control Panel
  - login.php          : Vendor Login Page
  - dashboard.php      : Vendor Dashboard (Manage products, view orders)

- customer/            : Customer Area
  - dashboard.php      : Customer Dashboard (View past orders)

- includes/            : Shared components (Header, Footer, DB Connection)

3. HOW TO SETUP & RUN
---------------------
1. Ensure XAMPP is installed and running (Apache & MySQL).
2. Create a MySQL database named 'nepal_milk_dairy'.
3. Import 'database.sql' into the database (optional, but recommended).
   - Note: 'index.php' contains logic to auto-create a default vendor account 
     and seed products if the database is empty/fresh.

5. TROUBLESHOOTING
------------------
- If "No products available" is displayed on the home page, refresh the page. 
  The first load attempts to seed the database.
- If login fails, check 'includes/db.php' to ensure database credentials (host, user, pass) 
  match your XAMPP configuration.
