### Setup Instructions
1. Install **XAMPP** and start Apache & MySQL.
2. Create a MySQL database: `nepal_milk_dairy`.
3. Import `database.sql` (optional; index.php auto-creates default vendor & products).
4. Access application via browser: `http://localhost/<your-folder>/index.php`.
